# group-encryption

This project is a personal learning exercise and is not suitable for general
use. Some of the code is partially written by LLMs. If for some reason you do
want to use it, go ahead!

A boring client-server application that I am making with the goal of being
robust and highly secure, even if relatively simple, so that I can learn more
about making the usual systems many companies develop.

More specifically, this is a rewrite of
[this annoying school project](https://github.com/noam2stein/school-project) I
had to write in Python, but now actually implemented well and written in Rust.
Its a simple project that still has many security concepts that I need to
implement correctly.
